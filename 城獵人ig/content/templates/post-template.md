# 📝 發文模板（霸氣鄰家大哥版）

---

## IG 爆文模板

```
【狂放標題】
（直戳痛點，不廢話）

---

【大哥開場】
（建立共鳴，像在巷口聊天）
例：「大哥我跟你講…」「這行水很深…」「實話告訴你…」

【口語文案】
（分段清晰，短句為主）
（多用反問句引起思考）
（引用法規時標註出處）

【互動誘餌】
（引導留言或私訊）
例：「留言 OOO，大哥教你」「想知道更多？追蹤起來」
```

---

## Threads 精簡版模板

```
（50-100 字，直接給重點）
（一個觀點 + 一個結論）
（行動呼籲）

#土地開發 #大哥講土地 #不動產
```

## 大哥風格用語庫

### 開場白
- 「大哥我跟你講」
- 「實話告訴你」
- 「這行水很深」
- 「這是血淚經驗」
- 「聽大哥的沒錯」

### 轉折語
- 「懂了嗎？」
- 「這就是重點——」
- 「所以說——」
- 「講白了就是——」

### 結尾語
- 「追蹤起來，大哥慢慢教」
- 「收藏起來，以後會用到」
- 「留言 OOO，大哥教你」
- 「不會的來問大哥」

### 禁止用語
- ❌ 首先、總之、此外、最後
- ❌ 作為一個、我能為您
- ❌ 無意義的 emoji 堆疊
- ❌ 用「您」（要用「你」）
